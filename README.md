# JARED-BRAVO
# Sistema de Registro de Usuarios

## Descripción
Proyecto: Sistema de registro de usuarios para la entrega del curso. Permite registrarse, verificar correo, iniciar sesión, recuperar contraseña y editar perfil.

## Objetivos
- Implementar un registro seguro de usuarios.
- Documentar requerimientos en DRS_v1 y DRS_v2.
- Probar funcionalidades críticas y proponer mantenimiento adaptativo.
